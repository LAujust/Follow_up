# Follow-Up For EP

## Project Overview
This repository contains code and resources for the Follow_up project. The purpose of this project is to coordinate follow-up network aimg to long-term monitoring transients detected by [Einstein Probe](https://ep.bao.ac.cn/).

## Features
- Download/Reduce Data from telescopes (which will be deprecated and merged to EP webpage);
- Automatic photometry for data from different facilities;
- Schedule and generate/send observation plans for telescopes;
- Quick view of targets and observation stats;
